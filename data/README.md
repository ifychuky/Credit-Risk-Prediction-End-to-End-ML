# Dataset Information

The dataset used for this project is the Loan Default Prediction dataset obtained from Kaggle.

Due to dataset ownership and reproducibility considerations, the raw dataset is not stored directly in this repository.

Dataset source:

[Loan Default Prediction Dataset - Kaggle](https://kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset)

Users can download the dataset from Kaggle and place it in this folder before running the notebook.
